# Luno Ludo

A standalone local Ludo game built with HTML, CSS, and JavaScript.

## Run

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Included

- Login-by-name screen
- 2 to 4 player game setup
- Bot players for non-human seats
- Dice turns
- Roll 6 to leave base
- Extra turn on 6
- Captures
- Safe squares
- Colored home lanes
- Win detection when all 4 tokens reach the center
